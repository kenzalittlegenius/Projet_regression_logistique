# Projet_regression_logistique


Logistic Regression from Scratch (Python)

This project consists of implementing a logistic regression model from scratch in Python, without using any machine learning libraries such as scikit-learn or TensorFlow.

The goal is to deeply understand the mathematical foundations and inner workings of a binary classification model, including optimization and interpretation.

Features

Manual implementation of:
Sigmoid function
Log-loss (cross-entropy) cost function
Gradient descent optimization
Data preprocessing:
Feature normalization
Train/test split
Model evaluation:
Accuracy
Confusion matrix
Precision, recall, F1-score
(Optional) ROC curve and AUC
Model interpretability:
Analysis and visualization of feature importance through learned weights

 Project Context

This project was developed as part of a Mathematics and Computer Science course, focusing on understanding how logistic regression works at a fundamental level.

Case Study

The model is applied to a simple real-world inspired problem:

Predicting university admission based on:

Test scores
High school performance
Extracurricular activities

The model outputs the probability of admission using a sigmoid function and learns parameters via gradient descent.

Key Concepts
Binary classification
Linear models
Optimization (gradient descent)
Probabilistic interpretation
Model interpretability

Deliverables
Fully commented Jupyter Notebook
Visualizations of feature importance
Performance evaluation metrics
Discussion on model limitations

Limitations
Assumes linear decision boundary
Sensitive to feature scaling
May underperform on complex datasets
